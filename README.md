# Mob timer

Real time peer to peer (WebRTC) mob programming timer built with [SvelteKit](https://kit.svelte.dev/) and [peer.js](https://peerjs.com/).

## Demo

https://oscargullberg.github.io/mob-timer/

## Run

1. `npm install`
2. `npm run dev`

## Verify

- `npm test`
- `npm run check`
- `npm run lint`
- `npm run build`
- `npm audit --audit-level=moderate`
